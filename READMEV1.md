# Job Application Assistant

AI-powered job search and application tracking tool with resume tailoring capabilities.

## Features
- Job application tracking with status management
- AI-powered resume tailoring via Claude API
- Job search integration
- Google Drive storage for resumes and data

## Setup
1. Open the app at: https://[your-username].github.io/job-application-assistant
2. Go to Settings tab
3. Configure your Make.com webhook URLs
4. Start tracking applications!

## Make.com Configuration Required
See setup instructions in the Settings tab of the application.

## Technologies
- React
- Tailwind CSS
- Make.com for API orchestration
- Google Drive for storage
- Claude API for AI features
